  <h1>:rocket: ig.news  </h1>
  <h2>Ignite React Track Project</h2>
  <p align="left">The project aims at studying and developing a ReactJS application with NextJS for listing posts and subscription system.</p>
  <p align="left">The application was developed using the Next.js framework applying concepts such as consuming external APIs, API Root, Server Side Rendering (SSR), Static Site Generation (SSG), STRIPE for subscription payments, NextAuth for authentication with Github, FaunaDB to store user information in a database, and Prismic CMS for adding and managing post content.</p>
</div>
